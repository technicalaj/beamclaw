Board not showing up, or it won't connect? Fix it →

Can't connect / no port shows up
  1. Use desktop Chrome or Edge — on a Mac too (Safari can't flash; nor can Firefox, iPhone/iPad or Android).
  2. Use a DATA USB cable. Many cheap cables are charge-only and show no port at all.
  3. Close the Arduino IDE and its Serial Monitor (or any serial terminal) — they hold the port, so the browser can't open it. Then unplug-replug and click Flash again.
  4. Clone board? Install its USB driver: CH340 or CP210x (most often needed on macOS).
  5. Still nothing? Try another USB port/cable, or use the IDE method below with the .hex.

② Then wire the light sensor

A 3-pin LDR module: S to A0, + to 5V, − to GND. Point it at your screen, then open the console and beam a behaviour.

Pins on cheap modules vary — match by function: signal/AO to A0, VCC/+ to 5V, GND/− to GND.

No desktop Chrome? Flash with the Arduino IDE

  1. Install the free Arduino IDE.
  2. Download the firmware: beamclaw_agent.ino
  3. Open it, choose Tools → Board → Arduino Uno + your port, click Upload.
  4. Serial Monitor @ 115200 should show == BeamClaw AGENT VM v2 ==. Wire the LDR, then beam a behaviour.

Advanced: prefer avrdude? Grab the compiled beamclaw_agent.hex.

In-browser flashing uses the Web Serial API (desktop Chrome/Edge). Your firmware never leaves your machine — it's pushed straight over USB.
